# MaterialTasksPlus

A material design To-do List application built from the ground up in Android.

Features:
 * Quick-add tasks
 * Remove task upon completion
 * Dark AMOLED theme
 * System notification settings
 
Coming-up:
 * Custom add tasks
 * Reminder notifications
 * Remove all tasks
 * Apply theme without restarting app
 * Save preferences
 * Stay tuned for more!
